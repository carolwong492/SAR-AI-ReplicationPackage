# Description of programs/code

<!---
Give a high-level overview of the program files and their purpose. Remove redundant/ obsolete files from the Replication archive.

The program files in this replication package contain the statistical tests used to explore the distribution of SAR patterns within Agentic AI pull requests, as well as their impacts on merge time and acceptance rates. 

- Files in `Data Analysis/` consist of the SAR pattern lists that compose the RegEx pattern used in the statistical tests.
- Files in `Statistical Tests/` will run the tests associated with each research question. Each individual file can be run in any sequence.
